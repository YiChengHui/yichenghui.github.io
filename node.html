Node.js

什么是node.js

  • 用js语言用来开发后台应用程序，它也是一个平台，和Web平台不一样

    Web平台:展示&交互

Node平台:
    1、给浏览器提供数据
        浏览器通过Ajax请求，后台给他的一般是JSON的数据
        访问首页，后台给你返回的就是整个网页(字符串)

    2、大量的数据统计

官网:
    https://nodejs.org/en/
    http://nodejs.cn/

Node.js和Web平台的js有啥不一样

  • 相似点:都是用js语法写的

  • 不同点:

    功能&职责不一样 ​

    REPL

  • 启动好的一个Node环境，这个时候我们把这个环境就称之为REPL环境， 有了这个环境之后我们就可以运行js代码

    退出 按住两次Ctrl+C，或者敲.exit

  • 当我们使用node去执行某个js文件的时候，流程如下
    1、在终端中输入 `node test.js`
2、开启一个REPL环境
3、使用node.exe去执行test.js中的代码
4、在cmd终端中输出一句话 `hello good girl!`
5、退出REPL环境

node全局对象的方式引用包

  • global 全局对象的方式引用包

    • other.js 在other中通过 global.a=100 将a暴露出去

    • app.js 然后在app.js 通过require的方式加载other（require('相对路径或者绝对路径')）

    • global暴露模块有一个问题是会污染全局变量

  • modules方式引用包

    • 通过modules.export将属性或者是方法暴露给外界

    • 外界通过require()的方式获取暴露出来的内容 let b=require('./other2.js)

    • other2.js module.exports={ b:b }

  • exports

    • 通过modules.export将属性或者是方法暴露给外界
    • 外界通过require()的方式获取暴露出来的内容 let b=require('./other2.js) b.b
    • other2.js exports.b=200

node使用fs模块读取文件

  • 异步读取

    const path = require('path') const fs = require('fs')

    //导入第三方包 var iconv = require('iconv-lite')

    //异步的方式去读取 fs.readFile(path.join(__dirname,'test.txt'),(err,data)=>{ const str = iconv.decode(data,'GBK') console.log(str) })

  • 同步读取

    const contents = fs.readFileSync(path.join(__dirname,'test.txt'));

    console.log(contents.toString())

  • node中异步和同步的区别：

    异步有回调函数，同步没有回调函数但是有返回值

node文件夹操作

  • 1.先引入path和fs这两个模块

    const path = require('path') const fs = require('fs')

  • //创建 文件夹必须我们自己创建

    fs.mkdir(path.join(__dirname,'abc'),(err)=>{ if(err){

       console.log(err)

    } console.log("create success") })

  • 判断文件夹是否存在 exists 不管可以判断文件夹还可以判断文件是否存在

    fs.exists(path.join(__dirname,'abc'),(exists)=>{ if(!exists){//不存在

       fs.mkdir(path.join(__dirname,'abc'),(err)=>{
       if(err){
           console.log(err)
       }            
       console.log("create success")
   })

    }else{

       console.log("文件夹已经存在...")

    } })

  • 重命名 rename 即可重命名文件夹也可重命名文件

    fs.exists(path.join(__dirname,'abc'),(exists)=>{ if(exists){//存在

       fs.rename(path.join(__dirname,'abc'),path.join(__dirname,'cba'),(err)=>{
       if(err){
           console.log(err)
       }

       console.log("rename OK")
   })

    }else{

       console.log("文件夹不存在")

    } })

  • 删除

    fs.exists(path.join(__dirname,'cba'),(exists)=>{ if(exists){//存在

       fs.rmdir(path.join(__dirname,'cba'),(err)=>{
       if(err){
           console.log(err)
       }
       console.log("remove OK")
   })

    }else{

       console.log("文件夹不存在")

    } })
