     var str = "123";
 var num =  Number(str);
 console.log(num); //123
 console.log(typeof num);//number

注意：
  1).如果字符串的值是一个数字，就可以转换成功，如果不是一个数字，就转换成NaN.
  2).如果字符串的值前后有空格，会忽略，如果中间有空格，会转换成NaN.
  3).如果是一个"",或者是一个" ",那么会转换成0.
  4).如果值是一个小数，也是能转换成小数的。

    a.2 使用 pasetInt(需要转换的内容);

 注意:
   1).从左往右，直到遇到一个非数字为止，前面的全部转换成功; 如果前面没有数字，就转化成NaN。
   2).如果是"",或者是" "，那么转化成NaN
   3).如果字符串的值是一个小数，那么得到的是小数的整数部分。

break和continue

  • 1.循环中的break 如果循环中遇到了break，那么就会结束这个循环。
  • 2.循环结束，有2种方式 a.循环的条件不满足，循环结束。 b.在循环里面遇见了break。
  • 3.使用break的场景？ 依次输入5个同学的成绩，输入错误就立即停止输入。 （1-100）
      var score;
  for(var i = 0 ; i < 5; i++){
      score = +prompt("请输入第"+(i+1)+"个同学的成绩");
      if(score < 1 || score > 100){
        break;
      }
  }

  • 4.循环中的continue 循环中如果遇见了continue，就会结束当前本次循环，当前本次循环continue后面的代码不会执行。注意整个循环并没有结束，整个循环要等到条件不成立，或者遇到break才会结束。

  • 5.continue的使用场景 计算1-100之间除了能被7整除之外所有整数的和。 var sum = 0; for(var i = 1 ; i <= 100; i++){ //不是吧所有的数都加进来，而是能被7整除的要除外不加。

          if(i % 7 == 0){
      continue;
      }
      sum += i;

    } console.log(sum);
